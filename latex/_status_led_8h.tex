\doxysubsection{Detailed Description}
Class to set an RGB LED to different colors based on the state of the system. 

Constructor\+: Status\+Led(int r\+\_\+pin, int g\+\_\+pin, int b\+\_\+pin) or (int r\+\_\+pin, int g\+\_\+pin, int b\+\_\+pin, int brightness) The pins are the RGB LED pins ideally they are PWM but can also handle simple digital pins. In the header set NO\+\_\+\+PWM to true or false depending on if you have PWM or simple digital pins. Brightness sets the brightness from 255 to 0, this is ignored for simple digital pins

update\+Led(int message) method sets the color of the LED, these messages can be found in the preprocessor part of the header. set\+Brightness(int brightness) method is used to change the brightness after initialization.
